# Super Store Analytics Dashboard
#Objective :
To design an interactive Super Store Analytics Dashboard in Power BI Desktop using the sample super store dataset.
##Dashboard Design & Insights :
Structured the report into multiple pages such as :
Executive summary page-KPI cards for Total sales, Total profit, orders, profit Margin, and Discount%.
Category & sub-category page- Bar charts, pie charts, and tree maps for top-performing categories and products.
Customer & segment page-sales and profit by segment and top-N customers.
Geographic Analysis page-Map and bar charts for Region - wise sales and profit.
